# 📈 Trading Advisor App

A comprehensive mobile trading advisor application that uses OCR from screenshots, technical/fundamental/sentiment analysis, and real-world event tracking to provide smart BUY/SELL/HOLD recommendations for any stock market.

## Key Features
- **OCR Screenshot Import**: Auto-extract price, ticker, volume, chart/indicator data
- **Real-Time News Integration**: Market, geopolitical, and company events
- **Multi-Strategy Analysis**: Strategies from top trading books and creators, weighted for accuracy
- **Performance Tracking**: Win rate, gains/losses, ROI, and recommendation accuracy history
- **Charts & Analytics Dashboard**: Track trends and app accuracy visually
- **Detailed, Beginner-Friendly Advice**: Risk level, entry/exit targets, and simple breakdowns for every recommendation

## Tech Stack
- React Native (mobile)
- Tesseract.js (OCR)
- News APIs, Alpha Vantage APIs
- Local storage (AsyncStorage)
- Redux, Chart libraries
